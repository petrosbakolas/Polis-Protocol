# Contributing to Polis Protocol Website

Thank you for your interest in contributing to the Polis Protocol website. We welcome contributions that align with our mission of "Ancient Wisdom, Digital Future."

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Language:** TypeScript
- **Icons:** Lucide React
- **Planned:** Rust (for various protocol logic and simulations)

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Run the development server: `npm run dev`.
4. Run linting: `npm run lint`.
5. Run tests (if available): `npm test`.

## Style Guidelines

- Follow the "Aegean Futurism" aesthetic (Deep Blue: `#0E2433`, Gold: `#D4AF37`).
- Use semantic HTML tags.
- Ensure type safety with TypeScript.
- Keep animations subtle and intentional (avoid rapid flashing or strobe effects).

## Pull Request Process

1. Create a new branch for your feature or fix.
2. Document your changes in the PR description.
3. Ensure the build passes: `npm run build`.
4. Request a review from the maintainers.
